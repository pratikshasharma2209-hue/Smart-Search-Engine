# Smart Search Engine

## Overview
A console-based Smart Search Engine developed in C++ using Object-Oriented Programming (OOP) and Data Structures & Algorithms.

## Features
- Add and display documents
- Inverted Index using Hash Map
- Keyword Search
- Multi-keyword Search
- Trie-based Auto Complete
- Search History
- Trending Searches
- Spell Checker (Edit Distance)
- File Handling for persistent storage

## Data Structures Used
- Vector
- Hash Map (unordered_map)
- Trie
- Queue
- Priority Queue
- Set

## Technologies
- C++
- OOP
- File Handling
- Data Structures & Algorithms

## How to Run

```bash
g++ main.cpp Document.cpp SearchEngine.cpp Trie.cpp History.cpp Trending.cpp SpellChecker.cpp -o SearchEngine.exe
./SearchEngine.exe
